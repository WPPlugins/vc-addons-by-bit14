A custom visual composer addon by bit14.com!

== Description ==

A collection of premium quality addons or extensions for Visual Composer page builder.

It features professional looking, easy to use yet highly functional extensions that can be used in a WPBakery Visual Composer page builder.

Visual Composer plugin must be activated to use this plugin. After you activate the required plugins, the elements should be available for use in Visual Composer.

= Features =

- Counters
- Progress Meter
- Iconic Lists
- Testimonials
- Newsletter Subscription

== Installation ==

1. Make sure visual composer(4.3 or higher) is already installed
2. Upload the plugin folder `bit14-vc-addons` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the "Plugins -> Installed 
plugins -> Activate Visual Composer Addons By Bit14" menu in WordPress.
4. Find Addons Tabs in Visual Composer Editor.


== Frequently Asked Questions ==

= How to use? =

Open Visual Composer Editor -> click on add element -> click on tab bit14 addons. click on any element you want and use it.

= Newsletter Subscription Not Working =

You must have installed and activated http://www.wordpress.org/plugins/email-subscribers to Newsletter Addon get worked.
